# 🤖 Rule-Based ChatBot

## Description
This project is a simple rule-based chatbot developed using Python for the CODSOFT Artificial Intelligence Internship.

## Features
- Greetings
- Date and Time Response
- AI Information
- Python Information
- Random Jokes
- Help Command
- Exit Command

## Technologies Used
- Python
- datetime
- random

## How to Run

```bash
python chatbot.py
```

## Sample Commands
- hi
- what is your name
- time
- date
- joke
- tell me 5 jokes
- what is ai
- python
- bye
