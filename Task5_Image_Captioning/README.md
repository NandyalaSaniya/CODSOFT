# Image Captioning System

## Description
A Python-based Image Captioning System that generates captions for images based on identified keywords in image names.

## Features
- Caption generation for multiple image categories
- User-friendly interface
- Handles unknown images with default captions
- Console-based implementation

## Technologies Used
- Python
- os module

## How to Run

```bash
python image_caption.py
```
